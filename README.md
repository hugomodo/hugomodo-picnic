# HugoModo Picnic

## Installation

```bash
git submodule add https://github.com/hugomodo/hugomodo.git themes/hugomodo
git submodule add https://github.com/hugomodo/hugomodo-picnic.git themes/hugomodo-picnic
```

```toml
themes = [
  "hugomodo-picnic"
]
```
